NATIVE

Bij native development leer je hoe je kunt programmeren voor Windows apparaten. Je leert onder andere de talen Python en C#. Op onze school maken we gebruik van Visual Studio, dat is een programmaomgeving waarin we programmeren. Hierin kan je met verschillende codetalen werken en daar maken we ook de applicaties mee. WEB

Tijdens de web development lessen leer je websites en webapplicaties maken. In het eerste blok leer je vooral front-end development, dat houdt in dat je met de voorkant van de website bezig bent (design), dit doen we met HTML en CSS. In het verloop van de opleiding komt er ook steeds meer back-end bij, hierbij moet je denken aan het koppelen van databases en externe informatie aan je front-end. Dit doen we met PHP en SQL. Je krijgt in dit vak veel vrijheid, bij elke opdracht mag je zelf bepalen hoe jij het gaat aanpakken.
